Welcome to the mods folder.

Pico's mods/Psych's mods folder.

If you are a beginner,
we recommend extracting the modTemplate.zip file to start developing your mod.

Read the Psych wiki on GitHub Here:
https://github.com/ShadowMario/FNF-PsychEngine/wiki
Read the Lua scripting API here:
https://shadowmario.github.io/psychengine.lua/

You can also add or edit specific files without the mod template,
just recreate the path to the desired file.

ABOUT EDITING:
If you want to change something, for example, in assets/shared/images or assets/base_game/images,
the edited files *must* be placed in mods/images. The engine will take care of the rest.
